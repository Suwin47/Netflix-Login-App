🎬 Netflix Login App

A full-stack Netflix-inspired Authentication Application built using React.js, Node.js, and Express.js. The application provides a clean Netflix-style login interface with user registration, login authentication, frontend validation, API integration, and dashboard redirection.

🚀 Features
Netflix-inspired responsive UI
User Registration (Sign Up)
User Login (Sign In)
Frontend form validation
Express.js backend API
Authentication using mock/local data
Persistent user storage using JSON file
Error handling for invalid credentials
Protected Dashboard page
React Router navigation
Fully responsive design
🛠️ Tech Stack
Frontend
React.js (Vite)
React Router DOM
CSS
Backend
Node.js
Express.js
CORS

⚡ Functionality
New users can create an account using the Sign Up page.
User data is stored in a JSON file.
Registered users can log in using their credentials.
Invalid credentials display appropriate error messages.
Successful login redirects users to the Dashboard page.
User data remains available even after server restarts.
